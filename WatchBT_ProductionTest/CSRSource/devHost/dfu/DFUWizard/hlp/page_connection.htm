Connection Type

The first item of information required by the wizard is the type of connection between the BlueCore module and your computer. Two options are offered:

Universal Serial Bus (USB)
Select this option if the BlueCore module is connected via a USB interface, either directly to the computer or indirectly via a hub.

COM port (RS-232)
Select this option if the BlueCore module is connected to an RS-232 COM port, or serial port, on your computer.

Select the appropriate option and then click on the Next > button to progress to the next page.
