Introduction
The BlueCore Device Firmware Upgrade (DFU) Wizard enables you to upgrade the firmware of a Cambridge Silicon Radio BlueCore module connected to your computer via USB or a COM port. It will take you through the following steps:
  • Upgrade the Firmware of your BlueCore Module
  • Connection Type
  • USB Device Selection
  • COM Port Selection
  • Upgrade Action
  • Upgrade File
  • Ready To Upgrade
  • Upgrade In Progress
  • Results
Use the above links or click Help on each page of the wizard to display information relevant to that page.

There is also a Command Line Interface.
